# 🤖 Caldinhas: Agente Financeiro Inteligente

O **Caldinhas** (Lucas) é um assistente financeiro focado em **organização de gastos**. Ele roda 100% localmente, garantindo total privacidade dos dados bancários do usuário. Ele não dá dicas de investimento, apenas analisa o padrão de consumo e ajuda na gestão do orçamento.

## 🚀 Tecnologias
- **Linguagem:** C# (.NET 8)
- **IA Local:** Ollama
- **Modelo:** `llama3.1:8b`

_Projeto desenvolvido para o desafio de Agentes Financeiros com IA Generativa._
